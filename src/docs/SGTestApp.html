Class: SGTestApp

SGTestApp()

Class of the webdevtest application. The app shows some promotion from a data feed in a list on the main page, and when the user picks one item from the list, it shows the detailed informations about the selected promotion. To work, it needs a div with 'containerDiv' id in the DOM.
